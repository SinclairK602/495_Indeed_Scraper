About

This web application utilizes the Entrez Direct API which is associated with PubMed to fetch Abstracts. Each day, the application will fetch the latest RCT and Meta-analysis studies and process them through a Natural language processing algorithm (NLP) to classify each sentence in the abstract to a respective category. Abstracts which do not exist in the database can be fetched using the search bar.

Please note: Abstracts that are not in the database will take some time to be fetched and processed. Please be patient.
